## Current Project :rocket:

**ETHU** — AI-powered Oral Health Management App
- Deep learning model for dental disease detection from panoramic X-ray (96% accuracy)
- Flutter cross-platform app (released on Google Play & App Store)
- Kotlin Spring backend deployed on AWS with Docker
- Features: AI X-ray analysis, oral health report, AI chatbot, nearby dentist finder, challenge system
